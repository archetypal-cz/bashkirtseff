# Translation Memory

This document serves as a translation memory for the Marie Bashkirtseff diary translation project. It contains key terms, phrases, and concepts that should be translated consistently throughout the project.

## Purpose

The translation memory helps ensure:
- Consistency across all translated documents
- Preservation of the author's voice and style
- Proper handling of period-specific terminology
- Accurate translation of names, places, and cultural references

## How to Use

When translating, refer to this document for established translations of recurring terms. If you encounter a term not listed here that you believe should be added, please add it with the following format:

```
### Term (French)
- Translation (Czech): [translation]
- Context: [brief explanation of the term and its context]
- First appearance: [document ID where it first appears]
- Notes: [any additional information about translation choices]
```
