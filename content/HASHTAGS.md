# HASHTAGS

This is an alphabetical list of hashtags used in the project, each linked to its corresponding glossary file in the `_glossary` directory. These hashtags represent people, places, events, and concepts mentioned in Marie Bashkirtseff's diary.

**IMPORTANT TAG RULES:**
- All hashtag filenames MUST be in CAPITAL_ASCII format (e.g., #MARIE_BASHKIRTSEFF)
- Use ONLY uppercase ASCII letters A-Z, numbers 0-9, and underscores
- NO accents or special characters (e.g., "Théâtre" becomes "THEATRE")
- NO lowercase letters in filenames
- Apostrophes, hyphens, and spaces become underscores
- This prevents duplicate files and encoding issues across platforms

When a new topic of interest appears in the diary, add it to this list and create a corresponding file in `/src/_original/_glossary/`.
