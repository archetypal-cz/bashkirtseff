### Key Design Decisions

1. **3 translators in parallel** — one per carnet, maximizing throughput
2. **OPS reviewers with 3 passes** — naturalness (no French), semantic accuracy (with French), deep linguistic check (Opus subagent)
3. **Russian influence detection** — explicit checklist of common russianisms provided to all agents
4. **One agent per carnet lifecycle** — fresh context prevents compaction failures
5. **No RSR/LAN agents** — source preparation already complete for all carnets

The ED applied frontmatter updates manually via sed. **Proposed fix**: Either give conductor subagent type Edit access, or document this as expected and have ED handle it.

## Issues Encountered

### Text Corruption (WATCHLIST: encoding issues)
- ops-007 found "баrilася" in 1873-07-14 — a mixed Cyrillic/Latin encoding corruption. Fixed to proper Ukrainian.

### False Friends (WATCHLIST: semantic shifts)
- "air" (expression/manner) translated as "повітря" (atmospheric air) in 008. Classic French→Ukrainian false friend caught by OPS.
- "Змалку" (since childhood) used where "З ранку" (in the morning) was intended in 006.

### Conductor Edit Access
- Conductors spawned as `conductor` subagent type lack Edit/Write tools. This is by design (read-only review) but means the ED must apply frontmatter changes. Not a bug, but worth documenting.
