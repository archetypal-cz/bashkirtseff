# Hashtags

This is an alphabetical list of hashtags used in the project, each linked to its corresponding glossary file in the `_glossary` directory. These hashtags represent people, places, events, and concepts mentioned in Marie Bashkirtseff's diary.

Hashtags must ONLY contain western ASCII characters and are not case sensitive. When a new topic of interest appears in the diary, add it to this list and create a corresponding file in `/src/_original/_glossary/`.
