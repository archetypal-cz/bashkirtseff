# Vendredi 2 novembre 1877
[//]: # ( 11.0542 )

[//]: # ( [#Paris](../_glossary/Paris.md) [#Blanc](../_glossary/Blanc.md) [#Cassagnac](../_glossary/Cassagnac.md) )

Blanc a dîné, on a parlé de Cassagnac d'une façon assez naturelle. Je voudrais bien savoir si Blanc sait ce que nous nous sommes écrit mon grand frère et moi.
[//]: # ( 11.0543 )

Nonobstant la présence du capitaine de frégate je me suis esquivée pour dessiner deux heures rue Vivienne.
[//]: # ( 11.0544 )

[//]: # ( [#Hall](../_glossary/Hall.md) )

J'ai répondu à Hall.
