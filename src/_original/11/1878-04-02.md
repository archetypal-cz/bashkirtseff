# Mardi 2 avril 1878
[//]: # ( 11.2479 )

[//]: # ( [#Paris](../_glossary/Paris.md) [#Marcuard](../_glossary/Marcuard.md) )

Les quatre voyageuses sont quatre poissons d'avril.
[//]: # ( 11.2480 )

Jeune harpiste.
[//]: # ( 11.2481 )

[//]: # ( [#Boyd](../_glossary/Boyd.md) [#comte_d_Osmond](../_glossary/comte_d_Osmond.md) [#La_Rotonde](../_glossary/La_Rotonde.md) )

Les Boyd ont invité plusieurs messieurs à dîner chez nous pour demain. C'est bête comme les Boyd. Il faudra les renvoyer
[//]: # ( 11.2482 )

puisque demain je vais à la "Rotonde" du comte d'Osmond. Vous savez-bien celui qui donne les plus belles fêtes en ce genre.
