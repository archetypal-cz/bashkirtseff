# Valsenter

**Research Status**: Basic
**Last Updated**: 2025-07-06
**Diary Coverage**: Up to 1874-03-18

## Identity
A tall woman in Nice society whom Marie calls an "obélisque" (obelisk).

## Diary References

### 1874
- **March 18**: Mentioned as example of Lambertye's type: "J'ai remarqué qu'il aime les obélisques, la Valsenter par exemple et plusieurs autres de ce genre" (I've noticed he likes obelisks, Valsenter for example and several others of this type).

## Context
- One of several tall women Lambertye admires
- Marie uses "obélisque" to describe tall/statuesque women
- Shows Lambertye's preference pattern
- Part of Nice social circle
